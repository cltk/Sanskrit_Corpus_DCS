jānapado 'bhijātaḥ svavagrahaḥ kṛtaśilpaścakṣuṣmān prājño dhārayiṣṇur dakṣo vāgmī pragalbhaḥ pratipattimān utsāhaprabhāvayuktaḥ kleśasahaḥ śucir maitro dṛḍhabhaktiḥ śīlabalārogyasattvayuktaḥ stambhacāpalahīnaḥ sampriyo vairāṇām akartetyamātyasampat // (1.1)
ataḥ pādārdhaguṇahīnau madhyamāvarau // (2.1)
teṣāṃ janapadabhijanam avagrahaṃ cāptataḥ parīkṣeta samānavidyebhyaḥ śilpaṃ śāstracakṣuṣmattāṃ ca karmārambheṣu prajñāṃ dhārayiṣṇutāṃ dākṣyaṃ ca kathāyogeṣu vāgmitvaṃ prāgalbhyaṃ pratibhānavattvaṃ ca saṃvāsibhyaḥ śīlabalārogyasattvayogam astambham acāpalaṃ ca pratyakṣataḥ sampriyatvam avairatvaṃ ca // (3.1)
pratyakṣaparokṣānumeyā hi rājavṛttiḥ // (4.1)
svayaṃ dṛṣṭaṃ pratyakṣam // (5.1)
paropadiṣṭaṃ parokṣam // (6.1)
karmasu kṛtenākṛtāvekṣaṇam anumeyam // (7.1)
yaugapadyāt tu karmaṇām anekatvād anekasthatvācca deśakālātyayo mā bhūd iti parokṣam amātyaiḥ kārayedityamātyakarma // (8.1)
purohitam uditoditakulaśīlaṃ sāṅge vede daive nimitte daṇḍanītyāṃ cābhivinītam āpadāṃ daivamānuṣīṇām atharvabhir upāyaiśca pratikartāraṃ kurvīta // (9.1)
tam ācāryaṃ śiṣyaḥ pitaraṃ putro bhṛtyaḥ svāminam iva cānuvarteta // (10.1)
brāhmaṇenaidhitaṃ kṣatraṃ mantrimantrābhimantritam / (11.1)
jayatyajitam atyantaṃ śāstrānugamaśastritam // (11.2)
