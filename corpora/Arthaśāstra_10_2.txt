grāmāraṇyānām adhvani niveśān yavasendhanodakavaśena parisaṃkhyāya sthānāsanagamanakālaṃ ca yātrāṃ yāyāt // (1.1)
tatpratīkāradviguṇaṃ bhaktopakaraṇaṃ vāhayet // (2.1)
aśakto vā sainyeṣvāyojayet antareṣu vā nicinuyāt // (3.1)
purastān nāyakaḥ madhye kalatraṃ svāmī ca pārśvayor aśvā bāhuutsāraḥ cakrānteṣu hastinaḥ prasāravṛddhir vā paścāt senāpatir yāyānniviśeta // (4.1)
sarvato vanājīvaḥ prasāraḥ // (5.1)
svadeśād anvāyatir vīvadhaḥ // (6.1)
mitrabalam āsāraḥ // (7.1)
kalatrasthānam apasāraḥ // (8.1)
purastād adhyāghāte makareṇa yāyāt paścācchakaṭena pārśvayor vajreṇa samantataḥ sarvatobhadreṇa ekāyane sūcyā // (9.1)
pathidvaidhībhāve svabhūmito yāyāt // (10.1)
abhūmiṣṭhānāṃ hi svabhūmiṣṭhā yuddhe pratilomā bhavanti // (11.1)
yojanam adhamā adhyardhaṃ madhyamā dviyojanam uttamā sambhāvyā vā gatiḥ // (12.1)
āśrayakārī sampannaghātī pārṣṇir āsāro madhyama udāsīno vā pratikartavyaḥ saṃkaṭo mārgaḥ śodhayitavyaḥ kośo daṇḍo mitrāmitrāṭavībalaṃ viṣṭiṛtur vā pratīkṣyāḥ kṛtadurgakarmanicayarakṣākṣayaḥ krītabalanirvedo mitrabalanirvedaścāgamiṣyati upajapitāro vā nātitvarayanti śatrur abhiprāyaṃ vā pūrayiṣyati iti śanair yāyāt viparyaye śīghram // (13.1)
hastistambhasaṃkramasetubandhanaukāṣṭhaveṇusaṃghātair alābucarmakaraṇḍadṛtiplavagaṇḍikāveṇikābhiścodakāni tārayet // (14.1)
tīrthābhigrahe hastyaśvair anyato rātrāv uttārya sattraṃ gṛhṇīyāt // (15.1)
anudake cakricatuṣpadaṃ cādhvapramāṇena śaktyodakaṃ vāhayet // (16.1)
dīrghakāntāram anudakaṃ yavasendhanodakahīnaṃ vā kṛcchrādhvānam abhiyogapraskannaṃ kṣutpipāsādhvaklāntaṃ paṅkatoyagambhīrāṇāṃ vā nadīdarīśailānām udyānāpayāne vyāsaktam ekāyanamārge śailaviṣame saṃkaṭe vā bahulībhūtaṃ niveśe prasthite visamnāhaṃ bhojanavyāsaktam āyatagatapariśrāntam avasuptaṃ vyādhimarakadurbhikṣapīḍitaṃ vyādhitapattyaśvadvipam abhūmiṣṭhaṃ vā balavyasaneṣu vā svasainyaṃ rakṣet parasainyaṃ cābhihanyāt // (17.1)
ekāyanamārgaprayātasya senāniścāragrāsāhāraśayyāprastārāgninidhānadhvajāyudhasaṃkhyānena parabalajñānam // (18.1)
tadātmāno gūhayet // (19.1)
