vaśiṣṭha uvāca / (1.1)
matsyādirūpiṇaṃ viṣṇuṃ brūhi sargādikāraṇam / (1.2)
purāṇaṃ brahma cāgneyaṃ yathā viṣṇoḥ purā śrutam // (1.3)
agnir uvāca / (2.1)
matsyāvatāraṃ vakṣye 'haṃ vasiṣṭha śṛṇu vai hareḥ / (2.2)
avatārakriyā duṣṭanaṣṭyai satpālanāya hi // (2.3)
āsīdatītakalpānte brāhmo naimittiko layaḥ / (3.1)
samudropaplutāstatra lokā bhūrādikā mune // (3.2)
manurvaivasvatastepe tapo vai bhuktimuktaye / (4.1)
ekadā kṛtamālāyāṃ kurvato jalatarpaṇaṃ // (4.2)
tasyāñjalyudake matsyaḥ svalpa eko 'bhyapadyata / (5.1)
kṣeptukāmaṃ jale prāha na māṃ kṣipa narottama // (5.2)
grāhādibhyo bhayaṃ me 'dya tac chrutvā kalaśe 'kṣipat / (6.1)
sa tu vṛddhaḥ punarmatsyaḥ prāha taṃ dehi me bṛhat // (6.2)
sthānametadvacaḥ śrutvā rājāthodañcane 'kṣipat / (7.1)
tatra vṛddho 'bravīdbhūpaṃ pṛthu dehi padaṃ mano // (7.2)
sarovare punaḥ kṣipto vavṛdhe tatpramāṇavān / (8.1)
ūce dehi bṛhat sthānaṃ prākṣipaccāmbudhau tataḥ // (8.2)
lakṣayojanavistīrṇaḥ kṣaṇamātreṇa so 'bhavat / (9.1)
matsyaṃ tamadbhutaṃ dṛṣṭvā vismitaḥ prābravīn manuḥ // (9.2)
ko bhavān nanu vai viṣṇur nārāyaṇa namo 'stu te / (10.1)
māyayā mohayasi māṃ kimarthaṃ tvaṃ janārdana // (10.2)
manunokto 'bravīnmatsyo manuṃ vai pālane ratam / (11.1)
avatīrṇo bhavāyāsya jagato duṣṭanaṣṭaye // (11.2)
saptame divase tv abdhiḥ plāvayiṣyati vai jagat / (12.1)
upasthitāyāṃ nāvi tvaṃ bījādīni vidhāya ca // (12.2)
saptarṣibhiḥ parivṛto niśāṃ brāhmīṃ cariṣyasi / (13.1)
upasthitasya me śṛṅge nibadhnīhi mahāhinā // (13.2)
ityuktvāntardadhe matsyo manuḥ kālapratīkṣakaḥ / (14.1)
sthitaḥ samudra udvele nāvamāruruhe tadā // (14.2)
ekaśṛṅgadharo matsyo haimo niyutayojanaḥ / (15.1)
nāvam babandha tacchṛṅge matsyākhyaṃ ca purāṇakam // (15.2)
śuśrāva matsyātpāpaghnaṃ saṃstuvan stutibhiś ca taṃ / (16.1)
brahmavedaprahartāraṃ hayagrīvaṃ ca dānavaṃ // (16.2)
avadhīt vedamatsyādyān pālayāmāsa keśavaḥ / (17.1)
prāpte kalpe 'tha vārāhe kūrmarūpo 'bhavaddhariḥ // (17.2)
ity ādimahāpurāṇe āgneye matsyāvatāro nāma dvitīyo 'dhyāyaḥ // (18.1)
