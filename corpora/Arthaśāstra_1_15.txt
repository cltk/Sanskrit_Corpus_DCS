mantrapūrvāḥ sarvārambhāḥ // (2.1)

tasmān mantroddeśam anāyukto nopagacchet // (4.1)
ucchidyeta mantrabhedī // (5.1)
mantrabhedo hi dūtāmātyasvāminām iṅgitākārābhyām // (6.1)
iṅgitam anyathāvṛttiḥ // (7.1)
ākṛtigrahaṇam ākāraḥ // (8.1)
tasya saṃvaraṇam āyuktapuruṣarakṣaṇam ā kāryakālād iti // (9.1)
teṣāṃ hi pramādamadasuptapralāpāḥ kāmādir utsekaḥ pracchanno 'vamato vā mantraṃ bhinatti // (10.1)
tasmād rakṣen mantram // (11.1)
mantrabhedo hyayogakṣemakaro rājñastadāyuktapuruṣāṇāṃ ca // (12.1)
tasmād guhyam eko mantrayeta iti bhāradvājaḥ // (13.1)
mantriṇām api hi mantriṇo bhavanti teṣām apyanye // (14.1)
saiṣā mantriparamparā mantraṃ bhinatti // (15.1)
tasmān nāsya pare vidyuḥ karma kiṃciccikīrṣitam / (16.1)
ārabdhārastu jānīyur ārabdhaṃ kṛtam eva vā // (16.2)
naikasya mantrasiddhir asti iti viśālākṣaḥ // (17.1)
pratyakṣaparokṣānumeyā hi rājavṛttiḥ // (18.1)
anupalabdhasya jñānam upalabdhasya niścitabalādhānam arthadvaidhasya saṃśayacchedanam ekadeśadṛṣṭasya śeṣopalabdhir iti mantrisādhyam etat // (19.1)
tasmād buddhivṛddhaiḥ sārdham adhyāsīta mantram // (20.1)
na kaṃcid avamanyeta sarvasya śṛṇuyān matam / (21.1)
bālasyāpyarthavadvākyam upayuñjīta paṇḍitaḥ // (21.2)
etan mantrajñānam naitan mantrarakṣaṇam iti pārāśarāḥ // (22.1)
yad asya kāryam abhipretaṃ tatpratirūpakaṃ mantriṇaḥ pṛcchet kāryam idam evam āsīt evaṃ vā yadi bhavet tat kathaṃ kartavyam iti // (23.1)
te yathā brūyustat kuryāt // (24.1)
evaṃ mantropalabdhiḥ saṃvṛtiśca bhavati iti // (25.1)
neti piśunaḥ // (26.1)
mantriṇo hi vyavahitam arthaṃ vṛttam avṛttaṃ vā pṛṣṭā anādareṇa bruvanti prakāśayanti vā // (27.1)
sa doṣaḥ // (28.1)
tasmāt karmasu ye yeṣvabhipretāstaiḥ saha mantrayeta // (29.1)
tair mantrayamāṇo hi mantrasiddhiṃ guptiṃ ca labhate iti // (30.1)
neti kauṭilyaḥ // (31.1)
anavasthā hyeṣā // (32.1)
mantribhistribhiścaturbhir vā saha mantrayeta // (33.1)
mantrayamāṇo hyekenārthakṛcchreṣu niścayaṃ nādhigacchet // (34.1)
ekaśca mantrī yatheṣṭam anavagrahaścarati // (35.1)
dvābhyāṃ mantrayamāṇo dvābhyāṃ saṃhatābhyām avagṛhyate vigṛhītābhyāṃ vināśyate // (36.1)
tat triṣu caturṣu vā kṛcchreṇopapadyate // (37.1)
mahādoṣam upapannaṃ tu bhavati // (38.1)
tataḥ pareṣu kṛcchreṇārthaniścayo gamyate mantro vā rakṣyate // (39.1)
deśakālakāryavaśena tvekena saha dvābhyām eko vā yathāsāmarthyaṃ mantrayeta // (40.1)
karmaṇām ārambhopāyaḥ puruṣadravyasampad deśakālavibhāgo vinipātapratīkāraḥ kāryasiddhir iti pañcāṅgo mantraḥ // (41.1)
tān ekaikaśaḥ pṛcchet samastāṃśca // (42.1)
hetubhiścaiṣāṃ matipravivekān vidyāt // (43.1)
avāptārthaḥ kālaṃ nātikrāmayet // (44.1)
na dīrghakālaṃ mantrayeta na teṣāṃ pakṣīyair yeṣām apakuryāt // (45.1)
mantripariṣadaṃ dvādaśāmātyān kurvīta iti mānavāḥ // (46.1)
ṣoḍaśa iti bārhaspatyāḥ // (47.1)
viṃśatim ityauśanasāḥ // (48.1)
yathāsāmarthyam iti kauṭilyaḥ // (49.1)
te hyasya svapakṣaṃ parapakṣaṃ ca cintayeyuḥ // (50.1)
akṛtārambham ārabdhānuṣṭhānam anuṣṭhitaviśeṣaṃ niyogasampadaṃ ca karmaṇāṃ kuryuḥ // (51.1)
āsannaiḥ saha karmāṇi paśyet // (52.1)
anāsannaiḥ saha pattrasampreṣaṇena mantrayeta // (53.1)
indrasya hi mantripariṣadṛṣīṇāṃ sahasram // (54.1)
sa taccakṣuḥ // (55.1)
tasmād imaṃ dvyakṣaṃ sahasrākṣam āhuḥ // (56.1)
ātyayike kārye mantriṇo mantripariṣadaṃ cāhūya brūyāt // (57.1)
tatra yadbhūyiṣṭhā brūyuḥ kāryasiddhikaraṃ vā tat kuryāt // (58.1)
kurvataśca / (59.1)
