amātyasampadopetāḥ sarvādhyakṣāḥ śaktitaḥ karmasu niyojyāḥ // (1.1)
karmasu caiṣāṃ nityaṃ parīkṣāṃ kārayet cittānityatvān manuṣyānām // (2.1)
aśvasadharmāṇo hi manuṣyā niyuktāḥ karmasu vikurvate // (3.1)
tasmāt kartāraṃ karaṇaṃ deśaṃ kālaṃ kāryaṃ prakṣepam udayaṃ caiṣu vidyāt // (4.1)
te yathāsaṃdeśam asaṃhatā avigṛhītāḥ karmāṇi kuryuḥ // (5.1)
saṃhatā bhakṣayeyuḥ vigṛhītā vināśayeyuḥ // (6.1)
na cānivedya bhartuḥ kaṃcid ārambhaṃ kuryuḥ anyatrāpatpratīkārebhyaḥ // (7.1)
pramādasthāneṣu caiṣām atyayaṃ sthāpayed divasavetanavyayadviguṇam // (8.1)
yaścaiṣāṃ yathādiṣṭam arthaṃ saviśeṣaṃ vā karoti sa sthānamānau labheta // (9.1)
alpāyatiścen mahāvyayo bhakṣayati // (10.1)
viparyaye yathāyativyayaśca na bhakṣayati ityācāryāḥ // (11.1)
apasarpeṇaivopalabhyeteti kauṭilyaḥ // (12.1)
yaḥ samudayaṃ parihāpayati sa rājārthaṃ bhakṣayati // (13.1)
sa ced ajñānādibhiḥ parihāpayati tad enaṃ yathāguṇaṃ dāpayet // (14.1)
yaḥ samudayaṃ dviguṇam udbhāvayati sa janapadaṃ bhakṣayati // (15.1)
sa ced rājārtham upanayatyalpāparādhe vārayitavyaḥ mahati yathāparādhaṃ daṇḍayitavyaḥ // (16.1)
yaḥ samudayaṃ vyayam upanayati sa puruṣakarmāṇi bhakṣayati // (17.1)
sa karmadivasadravyamūlyapuruṣavetanāpahāreṣu yathāparādhaṃ daṇḍayitavyaḥ // (18.1)
tasmād asya yo yasminn adhikaraṇe śāsanasthaḥ sa tasya karmaṇo yāthātathyam āyavyayau ca vyāsasamāsābhyām ācakṣīta // (19.1)
mūlaharatādātvikakadaryāṃśca pratiṣedhayet // (20.1)
yaḥ pitṛpaitāmaham artham anyāyena bhakṣayati sa mūlaharaḥ // (21.1)
yo yad yad utpadyate tat tad bhakṣayati sa tādātvikaḥ // (22.1)
yo bhṛtyātmapīḍābhyām upacinotyarthaṃ sa kadaryaḥ // (23.1)
sa pakṣavāṃśced anādeyaḥ viparyaye paryādātavyaḥ // (24.1)
yo mahatyarthasamudaye sthitaḥ kadaryaḥ saṃnidhatte 'vanidhatte 'vasrāvayati vā saṃnidhatte svaveśmani avanidhatte paurajānapadeṣu avasrāvayati paraviṣaye tasya sattrī mantrimitrabhṛtyabandhupakṣam āgatiṃ gatiṃ ca dravyāṇām upalabheta // (25.1)
yaścāsya paraviṣaye saṃcāraṃ kuryāt tam anupraviśya mantraṃ vidyāt // (26.1)
suvidite śatruśāsanāpadeśenainaṃ ghātayet // (27.1)
tasmād asyādhyakṣāḥ saṃkhyāyakalekhakarūpadarśakanīvīgrāhakottarādhyakṣasakhāḥ karmaṇi kuryuḥ // (28.1)
uttarādhyakṣā hastyaśvarathārohāḥ // (29.1)
teṣām antevāsinaḥ śilpaśaucayuktāḥ saṃkhyāyakādīnām apasarpāḥ // (30.1)
bahumukhyam anityaṃ cādhikaraṇaṃ sthāpayet // (31.1)
yathā hyanāsvādayituṃ na śakyaṃ jihvātalasthaṃ madhu vā viṣaṃ vā / (32.1)
arthastathā hyarthacareṇa rājñaḥ svalpo 'pyanāsvādayituṃ na śakyaḥ // (32.2)
matsyā yathāntaḥ salile caranto jñātuṃ na śakyāḥ salilaṃ pibantaḥ / (33.1)
yuktāstathā kāryavidhau niyuktā jñātuṃ na śakyā dhanam ādadānāḥ // (33.2)
api śakyā gatir jñātuṃ patatāṃ khe patatriṇām / (34.1)
na tu pracchannabhāvānāṃ yuktānāṃ caratāṃ gatiḥ // (34.2)
āsrāvayeccopacitān viparyasyecca karmasu / (35.1)
yathā na bhakṣayantyarthaṃ bhakṣitaṃ nirvamanti vā // (35.2)
na bhakṣayanti ye tvarthān nyāyato vardhayanti ca / (36.1)
nityādhikārāḥ kāryāste rājñaḥ priyahite ratāḥ // (36.2)
