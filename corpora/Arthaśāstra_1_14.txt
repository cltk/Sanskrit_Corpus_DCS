kṛtyākṛtyapakṣopagrahaḥ svaviṣaye vyākhyātaḥ paraviṣaye vācyaḥ // (1.1)
saṃśrutyārthān vipralabdhaḥ tulyakāriṇoḥ śilpe vopakāre vā vimānitaḥ vallabhāvaruddhaḥ samāhūya parājitaḥ pravāsopataptaḥ kṛtvā vyayam alabdhakāryaḥ svadharmād dāyādyād voparuddhaḥ mānādhikārābhyāṃ bhraṣṭaḥ kulyair antarhitaḥ prasabhābhimṛṣṭastrīkaḥ kārābhinyastaḥ paroktadaṇḍitaḥ mithyācāravāritaḥ sarvasvam āhāritaḥ bandhanaparikliṣṭaḥ pravāsitabandhuḥ iti kruddhavargaḥ // (2.1)
svayam upahataḥ viprakṛtaḥ pāpakarmābhikhyātaḥ tulyadoṣadaṇḍenodvignaḥ paryāttabhūmiḥ daṇḍenopanataḥ sarvādhikaraṇasthaḥ sahasopacitārthaḥ tatkulīno vāśaṃsuḥ pradviṣṭo rājñā rājadveṣī ca iti bhītavargaḥ // (3.1)
parikṣīṇaḥ anyāttasvaḥ kadaryaḥ vyasanī atyāhitavyavahāraśca iti lubdhavargaḥ // (4.1)
ātmasaṃbhāvitaḥ mānakāmaḥ śatrupūjāmarṣitaḥ nīcair upahitaḥ tīkṣṇaḥ sāhasikaḥ bhogenāsaṃtuṣṭaḥ iti mānivargaḥ // (5.1)
teṣāṃ muṇḍajaṭilavyañjanair yo yadbhaktiḥ kṛtyapakṣīyastaṃ tenopajāpayet // (6.1)
yathā madāndho hastī mattenādhiṣṭhito yad yad āsādayati tat sarvaṃ pramṛdnāti evam ayam aśāstracakṣur andho rājā paurajānapadavadhāyābhyutthitaḥ śakyam asya pratihastiprotsāhanenāpakartum amarṣaḥ kriyatām iti kruddhavargam upajāpayet // (7.1)
yathā līnaḥ sarpo yasmād bhayaṃ paśyati tatra viṣam utsṛjati evam ayaṃ rājā jātadoṣāśaṅkastvayi purā krodhaviṣam utsṛjati anyatra gamyatām iti bhītavargam upajāpayet // (8.1)
yathā śvagaṇināṃ dhenuḥ śvabhyo duhyate na brāhmaṇebhyaḥ evam ayaṃ rājā sattvaprajñāvākyaśaktihīnebhyo duhyate nātmaguṇasampannebhyaḥ asau rājā puruṣaviśeṣajñaḥ tatra gamyatām iti lubdhavargam upajāpayet // (9.1)
yathā caṇḍālodapānaścaṇḍālānām evopabhogyo nānyeṣām evam ayaṃ rājā nīco nīcānām evopabhogyo na tvadvidhānām āryāṇām asau rājā puruṣaviśeṣajñaḥ tatra gamyatām iti mānivargam upajāpayet // (10.1)
tatheti pratipannāṃstān saṃhitān paṇakarmaṇā / (11.1)
yojayeta yathāśakti sāpasarpān svakarmasu // (11.2)
labheta sāmadānābhyāṃ kṛtyāṃśca parabhūmiṣu / (12.1)
akṛtyān bhedadaṇḍābhyāṃ paradoṣāṃśca darśayan // (12.2)
