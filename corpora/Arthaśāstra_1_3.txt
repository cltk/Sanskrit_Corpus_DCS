sāmaṛgyajurvedās trayas trayī // (1.1)
atharvavedetihāsavedau ca vedāḥ // (2.1)
śikṣā kalpo vyākaraṇaṃ niruktaṃ chandovicitir jyotiṣam iti cāṅgāni // (3.1)
eṣa trayīdharmaścaturṇāṃ varṇānām āśramāṇāṃ ca svadharmasthāpanād aupakārikaḥ // (4.1)
svadharmo brāhmaṇasya adhyayanam adhyāpanaṃ yajanaṃ yājanaṃ dānaṃ pratigrahaśca // (5.1)
kṣatriyasyādhyayanaṃ yajanaṃ dānaṃ śastrājīvo bhūtarakṣaṇaṃ ca // (6.1)
vaiśyasyādhyayanaṃ yajanaṃ dānaṃ kṛṣipāśupālye vaṇijyā ca // (7.1)
śūdrasya dvijātiśuśrūṣā vārttā kārukuśīlavakarma ca // (8.1)
gṛhasthasya svadharmājīvastulyair asamānarṣibhir vaivāhyam ṛtugāmitvaṃ devapitratithipūjā bhṛtyeṣu tyāgaḥ śeṣabhojanaṃ ca // (9.1)
brahmacāriṇaḥ svādhyāyo 'gnikāryābhiṣekau bhaikṣavratitvam ācārye prāṇāntikī vṛttistadabhāve guruputre sabrahmacāriṇi vā // (10.1)
vānaprasthasya brahmacaryaṃ bhūmau śayyā jaṭājinadhāraṇam agnihotrābhiṣekau devatāpitratithipūjā vanyaścāhāraḥ // (11.1)
parivrājakasya jitendriyatvam anārambho niṣkiṃcanatvaṃ saṅgatyāgo bhaikṣavratam anekatrāraṇye ca vāso bāhyābhyantaraṃ ca śaucam // (12.1)
sarveṣām ahiṃsā satyaṃ śaucam anasūyānṛśaṃsyaṃ kṣamā ca // (13.1)
svadharmaḥ svargāyānantyāya ca // (14.1)
tasyātikrame lokaḥ saṃkarād ucchidyeta // (15.1)
tasmāt svadharmaṃ bhūtānāṃ rājā na vyabhicārayet / (16.1)
svadharmaṃ saṃdadhāno hi pretya ceha ca nandati // (16.2)
vyavasthitāryamaryādaḥ kṛtavarṇāśramasthitiḥ / (17.1)
trayyābhirakṣito lokaḥ prasīdati na sīdati // (17.2)
