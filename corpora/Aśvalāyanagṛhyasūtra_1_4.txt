 prayāṇa upapadyamāne pūṣā tveto nayatu hastagṛhyeti yānam ārohayet // (1.1)
aśmanvatīr īyate saṃrabhadhvam ity ardharcena nāvam ārohayet // (2.1)
uttareṇotkramayet // (3.1)
jīvaṃ rudantīti rudatyām // (4.1)
vivāhāgnim agrato 'jasraṃ nayanti // (5.1)
kalyāṇeṣu deśavṛkṣacatuṣpatheṣu mā vidan paripanthina iti japet // (6.1)
vāse vāse sumaṅgalīr iyaṃ vadhūr itīkṣakān īkṣeta // (7.1)
iha priyaṃ prajayā te samṛdhyatām iti gṛhaṃ praveśayet // (8.1)
vivāhāgnim upasamādhāya paścād asyānaḍuhaṃ carmāstīrya prāggrīvam uttaraloma tasminn upaviṣṭāyāṃ samanvārabdhāyām ā naḥ prajāṃ janayatu prajāpatir iti catasṛbhiḥ pratyṛcaṃ hutvā samañjantu viśve devā iti dadhnaḥ prāśya pratiprayacched ājyaśeṣeṇa vānakti hṛdaye // (9.1)
ata ūrdhvam akṣārālavaṇāśinau brahmacāriṇāv alaṃkurvāṇāv adhaḥśāyinau syātām trirātraṃ dvādaśarātram // (10.1)
saṃvatsaraṃ vaika ṛṣir jāyata iti // (11.1)
caritavrataḥ sūryāvide vadhūvastraṃ dadyāt // (12.1)
annaṃ brāhmaṇebhyaḥ // (13.1)
atha svastyayanaṃ vācayīta // (14.1)
