saṃnidhātā kośagṛhaṃ paṇyagṛhaṃ koṣṭhāgāraṃ kupyagṛham āyudhāgāraṃ bandhanāgāraṃ ca kārayet // (1.1)
caturaśrāṃ vāpīm anudakopasnehāṃ khānayitvā pṛthuśilābhir ubhayataḥ pārśvaṃ mūlaṃ ca pracitya sāradārupañjaraṃ bhūmisamaṃ tritalam anekavidhānaṃ kuṭṭimadeśasthānatalam ekadvāraṃ yantrayuktasopānaṃ bhūmigṛhaṃ kārayet // (2.1)
tasyoparyubhayatoniṣedhaṃ sapragrīvam aiṣṭakaṃ bhāṇḍavāhinīparikṣiptaṃ kośagṛhaṃ kārayet prāsādaṃ vā // (3.1)
janapadānte dhruvanidhim āpadartham abhityaktaiḥ kārayet // (4.1)
pakveṣṭakāstambhaṃ catuḥśālam ekadvāram anekasthānatalaṃ vivṛtastambhāpasāram ubhayataḥ paṇyagṛhaṃ koṣṭhāgāraṃ ca dīrghabahuśālaṃ kakṣyāvṛtakuḍyam antaḥ kupyagṛham tad eva bhūmigṛhayuktam āyudhāgāraṃ pṛthagdharmasthīyaṃ mahāmātrīyaṃ vibhaktastrīpuruṣasthānam apasārataḥ suguptakakṣyaṃ bandhanāgāraṃ kārayet // (5.1)
sarveṣāṃ śālāḥ khātodapānavacca snānagṛhāgniviṣatrāṇamārjāranakulārakṣāsvadaivatapūjanayuktāḥ kārayet // (6.1)
koṣṭhāgāre varṣamānam aratnimukhaṃ kuṇḍaṃ sthāpayet // (7.1)
tajjātakaraṇādhiṣṭhitaḥ purāṇaṃ navaṃ ca ratnaṃ sāraṃ phalgu kupyaṃ vā pratigṛhṇīyāt // (8.1)
tatra ratnopadhāv uttamo daṇḍaḥ kartuḥ kārayituśca sāropadhau madhyamaḥ phalgukupyopadhau tacca tāvacca daṇḍaḥ // (9.1)
rūpadarśakaviśuddhaṃ hiraṇyaṃ pratigṛhṇīyāt // (10.1)
aśuddhaṃ chedayet // (11.1)
āhartuḥ pūrvaḥ sāhasadaṇḍaḥ // (12.1)
śuddhaṃ pūrṇam abhinavaṃ ca dhānyaṃ pratigṛhṇīyāt // (13.1)
viparyaye mūlyadviguṇo daṇḍaḥ // (14.1)
tena paṇyaṃ kupyam āyudhaṃ ca vyākhyātam // (15.1)
sarvādhikaraṇeṣu yuktopayuktatatpuruṣāṇāṃ paṇādicatuṣpaṇaparamāpahāreṣu pūrvamadhyamottamavadhā daṇḍāḥ // (16.1)
kośādhiṣṭhitasya kośāvacchede ghātaḥ // (17.1)
tadvaiyāvṛtyakarāṇām ardhadaṇḍāḥ // (18.1)
paribhāṣaṇam avijñāte // (19.1)
corāṇām abhipradharṣaṇe citro ghātaḥ // (20.1)
tasmād āptapuruṣādhiṣṭhitaḥ saṃnidhātā nicayān anutiṣṭhet // (21.1)
bāhyam abhyantaraṃ cāyaṃ vidyād varṣaśatād api / (22.1)
yathā pṛṣṭo na sajjeta vyaye śeṣe ca saṃcaye // (22.2)
