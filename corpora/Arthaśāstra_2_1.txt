bhūtapūrvam abhūtapūrvaṃ vā janapadaṃ paradeśāpavāhanena svadeśābhiṣyandavamanena vā niveśayet // (1.1)
śūdrakarṣakaprāyaṃ kulaśatāvaraṃ pañcakulaśataparaṃ grāmaṃ krośadvikrośasīmānam anyonyārakṣaṃ niveśayet // (2.1)
nadīśailavanabhṛṣṭidarīsetubandhaśamīśālmalīkṣīravṛkṣān anteṣu sīmnāṃ sthāpayet // (3.1)
aṣṭaśatagrāmyā madhye sthānīyam catuḥśatagrāmyā droṇamukham dviśatagrāmyāḥ kārvaṭikam daśagrāmīsaṃgraheṇa saṃgrahaṃ sthāpayet // (4.1)
anteṣvantapāladurgāṇi janapadadvārāṇyantapālādhiṣṭhitāni sthāpayet // (5.1)
teṣām antarāṇi vāgurikaśabarapulindacaṇḍālāraṇyacarā rakṣeyuḥ // (6.1)
ṛtvigācāryapurohitaśrotriyebhyo brahmadeyāny adaṇḍakarāṇy abhirūpadāyādakāni prayacchet adhyakṣasaṃkhyāyakādibhyo gopasthānikānīkasthacikitsakāśvadamakajaṅghākārikebhyaśca vikrayādhānavarjāni // (7.1)
karadebhyaḥ kṛtakṣetrāṇyaikapuruṣikāṇi prayacchet // (8.1)
akṛtāni kartṛbhyo nādeyāni // (9.1)
akṛṣatām ācchidyānyebhyaḥ prayacchet // (10.1)
grāmabhṛtakavaidehakā vā kṛṣeyuḥ // (11.1)
akṛṣanto vāvahīnaṃ dadyuḥ // (12.1)
dhānyapaśuhiraṇyaiścaitān anugṛhṇīyāt // (13.1)
tānyanu sukhena dadyuḥ // (14.1)
anugrahaparihārau caitebhyaḥ kośavṛddhikarau dadyāt kośopaghātakau varjayet // (15.1)
alpakośo hi rājā paurajānapadān eva grasate // (16.1)
niveśasamakālaṃ yathāgatakaṃ vā parihāraṃ dadyāt // (17.1)
nivṛttaparihārān pitevānugṛhṇīyāt // (18.1)
ākarakarmāntadravyahastivanavrajavaṇikpathapracārān vāristhalapathapaṇyapattanāni ca niveśayet // (19.1)
sahodakam āhāryodakaṃ vā setuṃ bandhayet // (20.1)
anyeṣāṃ vā badhnatāṃ bhūmimārgavṛkṣopakaraṇānugrahaṃ kuryāt puṇyasthānārāmāṇāṃ ca // (21.1)
sambhūya setubandhād apakrāmataḥ karmakarabalīvardāḥ karma kuryuḥ // (22.1)
vyayakarmaṇi ca bhāgī syāt na cāṃśaṃ labheta // (23.1)
matsyaplavaharitapaṇyānāṃ setuṣu rājā svāmyaṃ gacchet // (24.1)
dāsāhitakabandhūn aśṛṇvato rājā vinayaṃ grāhayet // (25.1)
bālavṛddhavyasanyanāthāṃśca rājā bibhṛyāt striyam aprajātāṃ prajātāyāśca putrān // (26.1)
bāladravyaṃ grāmavṛddhā vardhayeyur ā vyavahāraprāpaṇāt devadravyaṃ ca // (27.1)
apatyadāraṃ mātāpitarau bhrātṝn aprāptavyavahārān bhaginīḥ kanyā vidhavāścābibhrataḥ śaktimato dvādaśapaṇo daṇḍaḥ anyatra patitebhyaḥ anyatra mātuḥ // (28.1)
putradāram apratividhāya pravrajataḥ pūrvaḥ sāhasadaṇḍaḥ striyaṃ ca pravrājayataḥ // (29.1)
luptavyāyāmaḥ pravrajed āpṛcchya dharmasthān // (30.1)
anyathā niyamyeta // (31.1)
vānaprasthād anyaḥ pravrajitabhāvaḥ sajātād anyaḥ saṃghaḥ sāmutthāyikād anyaḥ samayānubandho vā nāsya janapadam upaniviśeta // (32.1)
na ca tatrārāmā vihārārthā vā śālāḥ syuḥ // (33.1)
naṭanartakagāyanavādakavāgjīvanakuśīlavā na karmavighnaṃ kuryuḥ // (34.1)
nirāśrayatvād grāmāṇāṃ kṣetrābhiratatvācca puruṣāṇāṃ kośaviṣṭidravyadhānyarasavṛddhir bhavati // (35.1)
paracakrāṭavīgrastaṃ vyādhidurbhikṣapīḍitam / (36.1)
deśaṃ parihared rājā vyayakrīḍāśca vārayet // (36.2)
daṇḍaviṣṭikarābādhai rakṣed upahatāṃ kṛṣim / (37.1)
stenavyālaviṣagrāhair vyādhibhiśca paśuvrajān // (37.2)
vallabhaiḥ kārmikaiḥ stenair antapālaiśca pīḍitam / (38.1)
śodhayet paśusaṃghaiśca kṣīyamāṇaṃ vaṇikpatham // (38.2)
evaṃ dravyadvipavanaṃ setubandham athākarān / (39.1)
rakṣet pūrvakṛtān rājā navāṃścābhipravartayet // (39.2)
