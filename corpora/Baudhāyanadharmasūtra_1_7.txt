 kamaṇḍalur dvijātīnāṃ śaucārthaṃ vihitaḥ purā / (1.1)
brahmaṇā munimukhyaiś ca tasmāt taṃ dhārayet sadā // (1.2)
tataḥ śaucaṃ tataḥ pānaṃ saṃdhyopāsanam eva ca / (2.1)
nirviśaṅkena kartavyaṃ yadīcchecchreya ātmanaḥ // (2.2)
kuryācchuddhena manasā na cittaṃ dūṣayed budhaḥ sahakamaṇḍalunotpannaḥ svayaṃbhūs tasmāt kamaṇḍalunā caret // (3.1)
mūtrapurīṣe kurvan dakṣiṇe haste gṛhṇāti savya ācamanīyam etat sidhyati sādhūnām // (4.1)
yathā hi somasaṃyogāc camaso medhya ucyate / (5.1)
apāṃ tathaiva saṃyogān nityo medhyaḥ kamaṇḍaluḥ // (5.2)
pitṛdevāgnikāryeṣu tasmāt taṃ parivarjayet // (6.1)
tasmād vinā kamaṇḍalunā nādhvānaṃ vrajen na sīmantaṃ nagṛhād gṛham // (7.1)
padam api na gacched iṣumātrād ity eke // (8.1)
yad icched dharmasaṃtatim iti baudhāyanaḥ // (9.1)
ṛgvidheneti vāg vadati ṛgvidheneti vāg vadati // (10.1)
