surādhyakṣaḥ surākiṇvavyavahārān durge janapade skandhāvāre vā tajjātasurākiṇvavyavahāribhiḥ kārayed ekamukham anekamukhaṃ vā vikrayakrayavaśena vā // (1.1)
ṣaṭśatam atyayam anyatra kartṛkretṛvikretṝṇāṃ sthāpayet // (2.1)
grāmād anirṇayaṇam asaṃpātaṃ ca surāyāḥ pramādabhayāt karmasu nirdiṣṭānām maryādātikramabhayād āryāṇām utsāhabhayācca tīkṣṇānām // (3.1)
lakṣitam alpaṃ vā caturbhāgam ardhakuḍubaṃ kuḍubam ardhaprasthaṃ prasthaṃ veti jñātaśaucā nirhareyuḥ // (4.1)
pānāgāreṣu vā pibeyur asaṃcāriṇaḥ // (5.1)
nikṣepopanidhiprayogāpahṛtānām aniṣṭopagatānāṃ ca dravyāṇāṃ jñānārtham asvāmikaṃ kupyaṃ hiraṇyaṃ copalabhya nikṣeptāram anyatra vyapadeśena grāhayed ativyayakartāram anāyativyayaṃ ca // (6.1)
na cānargheṇa kālikāṃ vā surāṃ dadyād anyatra duṣṭasurāyāḥ // (7.1)
tām anyatra vikrāpayet // (8.1)
dāsakarmakarebhyo vā vetanaṃ dadyāt // (9.1)
vāhanapratipānaṃ sūkarapoṣaṇaṃ vā dadyāt // (10.1)
pānāgārāṇyanekakakṣyāṇi vibhaktaśayanāsanavanti pānoddeśāni gandhamālyodakavanti ṛtusukhāni kārayet // (11.1)
tatrasthāḥ prakṛtyautpattikau vyayau gūḍhā vidyuḥ āgantūṃśca // (12.1)
kretṝṇāṃ mattasuptānām alaṅkārācchādanahiraṇyāni ca vidyuḥ // (13.1)
tannāśe vaṇijastacca tāvacca daṇḍaṃ dadyuḥ // (14.1)
vaṇijastu saṃvṛteṣu kakṣyāvibhāgeṣu svadāsībhiḥ peśalarūpābhir āgantūnāṃ vāstavyānāṃ cāryarūpāṇāṃ mattasuptānāṃ bhāvaṃ vidyuḥ // (15.1)
medakaprasannāsavāriṣṭamaireyamadhūnām // (16.1)
udakadroṇaṃ taṇḍulānām ardhāḍhakaṃ trayaḥ prasthāḥ kiṇvasyeti medakayogaḥ // (17.1)
dvādaśāḍhakaṃ piṣṭasya pañca prasthāḥ kiṇvasya kramukatvakphalayukto vā jātisambhāraḥ prasannāyogaḥ // (18.1)
kapitthatulā phāṇitaṃ pañcataulikaṃ prastho madhuna ityāsavayogaḥ // (19.1)
pādādhiko jyeṣṭhaḥ pādahīnaḥ kaniṣṭhaḥ // (20.1)
cikitsakapramāṇāḥ pratyekaśo vikārāṇām ariṣṭāḥ // (21.1)
meṣaśṛṅgītvakkvāthābhiṣuto guḍapratīvāpaḥ pippalīmaricasambhārastriphalāyukto vā maireyaḥ // (22.1)
guḍayuktānāṃ vā sarveṣāṃ triphalāsambhāraḥ // (23.1)
mṛdvīkāraso madhu // (24.1)
tasya svadeśo vyākhyānaṃ kāpiśāyanaṃ hārahūrakam iti // (25.1)
māṣakalanīdroṇamāmaṃ siddhaṃ vā tribhāgādhikataṇḍulaṃ moraṭādīnāṃ kārṣikabhāgayuktaṃ kiṇvabandhaḥ // (26.1)
pāṭhālodhratejovatyelāvālukamadhukamadhurasāpriyaṅgudāruharidrāmaricapippalīnāṃ ca pañcakārṣikaḥ sambhārayogo medakasya prasannāyāśca // (27.1)
madhukaniryūhayuktā kaṭaśarkarā varṇaprasādanī ca // (28.1)
cocacitrakavilaṅgagajapippalīnāṃ ca kārṣikaḥ kramukamadhukamustālodhrāṇāṃ dvikārṣikaścāsavasambhāraḥ // (29.1)
daśabhāgaścaiṣāṃ bījabandhaḥ // (30.1)
prasannāyogaḥ śvetasurāyāḥ // (31.1)
sahakārasurā rasottarā bījottarā vā mahāsurā saṃbhārikī vā // (32.1)
tāsāṃ moraṭāpalāśapattūrameṣaśṛṅgīkarañjakṣīravṛkṣakaṣāyabhāvitaṃ dagdhakaṭaśarkarācūrṇaṃ lodhracitrakavilaṅgapāṭhāmustākaliṅgayavadāruharidrendīvaraśatapuṣpāpāmārgasaptaparṇanimbāsphotakalkārdhayuktam antarnakho muṣṭiḥ kumbhīṃ rājapeyāṃ prasādayati // (33.1)
phāṇitaḥ pañcapalikaścātra rasavṛddhir deyaḥ // (34.1)
kuṭumbinaḥ kṛtyeṣu śvetasurām auṣadhārthaṃ vāriṣṭam anyad vā kartuṃ labheran // (35.1)
utsavasamājayātrāsu caturahaḥ sauriko deyaḥ // (36.1)
teṣvananujñātānāṃ prahavanāntaṃ daivasikam atyayaṃ gṛhṇīyāt // (37.1)
surākiṇvavicayaṃ striyo bālāśca kuryuḥ // (38.1)
arājapaṇyāḥ pañcakaṃ śataṃ śulkaṃ dadyuḥ surakāmedakāriṣṭamadhuphalāmlāmlaśīdhūnāṃ ca // (39.1)
ahnaśca vikrayaṃ jñātvā vyājīṃ mānahiraṇyayoḥ / (40.1)
tathā vaidharaṇaṃ kuryād ucitaṃ cānuvartayet // (40.2)
