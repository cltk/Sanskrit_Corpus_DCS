 nārada uvāca / (1.1)
rāmo vaśiṣṭhaṃ mātṝṃśca natvātiṃ ca praṇamya saḥ / (1.2)
anasūyāṃ ca tatpatnīṃ śarabhaṅgaṃ sutīkṣṇakam // (1.3)
agastyabhrātaraṃ natvā agastyaṃ tatprasādataḥ / (2.1)
dhanuḥkhaḍgaṃ ca samprāpya daṇḍakāraṇyamāgataḥ // (2.2)
janasthāne pañcavaṭyāṃ sthito godāvarītaṭe / (3.1)
tatra śūrpaṇakhāyātā bhakṣituṃ tān bhayaṅkarī // (3.2)
rāmaṃ surūpaṃ dṛṣṭvā sā kāminī vākyamabravīt / (4.1)
kastvaṃ kasmātsamāyāto bhartā me bhava cārthitaḥ // (4.2)
etau ca bhakṣayiṣyāmi ity uktvā taṃ samudyatā / (5.1)
tasyā nāsāṃ ca karṇau ca rāmokto lakṣmaṇo 'chinat // (5.2)
raktaṃ kṣarantī prayayau kharaṃ bhrātaramabravīt / (6.1)
mariṣyāmi vinā sāhaṃ khara jīvāmi vai tadā // (6.2)
rāmasya bhāryā sītāsau tasyāsīllakṣmaṇo 'nujaḥ / (7.1)
teṣām yadrudhiraṃ soṣṇaṃ pāyayiṣyasi māṃ yadi // (7.2)
kharastatheti tāmuktvā caturdaśasahasrakaiḥ / (8.1)
rakṣasāṃ dūṣaṇenāgād yoddhuṃ triśirasā saha // (8.2)
rāmaṃ rāmo 'pi yuyudhe śarair vivyādha rākṣasān / (9.1)
hastyaśvarathapādātaṃ balaṃ ninye yamakṣayaṃ // (9.2)
triśīrṣāṇaṃ kharaṃ raudraṃ yudhyantaṃ caiva dūṣaṇam / (10.1)
yayau śūrpaṇakhā laṅkāṃ rāvaṇāgre 'patad bhuvi // (10.2)
abravīdrāvaṇaṃ kruddhā na tvaṃ rājā na rakṣakaḥ / (11.1)
kharādihantū rāmasya sītāṃ bhāryāṃ harasva ca // (11.2)
rāmalakṣmaṇaraktasya pānājjīvāmi nānyathā / (12.1)
tathetyāha ca tac chrutvā mārīcaṃ prāha vai vraja // (12.2)
svarṇacitramṛgo bhūtvā rāmalakṣmaṇakarṣakaḥ / (13.1)
sītāgre tāṃ hariṣyāmi anyathā maraṇaṃ tava // (13.2)
mārīco rāvaṇaṃ prāha rāmo mṛtyurdhanurdharaḥ / (14.1)
rāvaṇādapi martavyaṃ martavyaṃ rāghavādapi // (14.2)
avaśyaṃ yadi martavyaṃ varaṃ rāmo na rāvaṇaḥ / (15.1)
iti matvā mṛgo bhūtvā sītāgre vyacaranmuhuḥ // (15.2)
sītayā prerito rāmaḥ śareṇāthāvadhīcca taṃ / (16.1)
mriyamāṇo mṛgaḥ prāha hā sīte lakṣmaṇeti ca // (16.2)
saumitriḥ sītayokto 'tha viruddhaṃ rāmamāgataḥ / (17.1)
rāvaṇo 'pyaharat sītāṃ hatvā gṛdhraṃ jaṭāyuṣaṃ // (17.2)
jaṭāyuṣā sa bhinnāṅgo 'ṅkenādāya jānakīm / (18.1)
gato laṅkāmaśokākhye dhārayāmāsa cābravīt // (18.2)
bhava bhāryā mamāgryā tvaṃ rākṣasyo rakṣyatāmiyam / (19.1)
rāmo hatvā tu mārīcaṃ dṛṣṭvā lakṣmaṇamabravīt // (19.2)
māyāmṛgo 'sau saumitre yathā tvamiha cāgataḥ / (20.1)
tathā sītā hṛtā nūnaṃ nāpaśyat sa gato 'tha tām // (20.2)
śuśoca vilalāpārto māṃ tyaktvā kva gatāsi vai / (21.1)
lakṣmaṇāśvāsito rāmo mārgayāmāsa jānakīm // (21.2)
dṛṣṭvā jaṭāyustaṃ prāha rāvaṇo hṛtavāṃś ca tāṃ / (22.1)
mṛto 'tha saṃskṛtastena kabandhaṃ cāvadhīttataḥ // (22.2)
śāpamukto 'bravīdrāmaṃ sa tvaṃ sugrīvamāvraja // (23.1)
ity ādimahāpurāṇe āgneye rāmāyaṇe āraṇyakakāṇḍavarṇanaṃ nāma saptamo 'dhyāyaḥ // (24.1)
