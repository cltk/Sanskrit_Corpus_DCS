vāstukapraśaste vāstuni nāyakavardhakimauhūrtikāḥ skandhāvāram vṛttaṃ dīrghaṃ caturaśraṃ vā bhūmivaśena vā caturdvāraṃ ṣaṭpathaṃ navasaṃsthānaṃ māpayeyuḥ khātavaprasāladvārāṭṭālakasampannaṃ bhaye sthāne ca // (1.1)
madhyamasyottare navabhāge rājavāstukaṃ dhanuḥśatāyāmam ardhavistāram paścimārdhe tasyāntaḥpuram // (2.1)
antarvaṃśikasainyaṃ cānte niviśeta // (3.1)
purastād upasthānam dakṣiṇataḥ kośaśāsanakāryakaraṇāni vāmato rājopavāhyānāṃ hastyaśvarathānāṃ sthānam // (4.1)
ato dhanuḥśatāntarāścatvāraḥ śakaṭamethīpratatistambhasālaparikṣepāḥ // (5.1)
prathame purastān mantripurohitau dakṣiṇataḥ koṣṭhāgāraṃ mahānasaṃ ca vāmataḥ kupyāyudhāgāram // (6.1)
dvitīye maulabhṛtānāṃ sthānam aśvarathānāṃ senāpateśca // (7.1)
tṛtīye hastinaḥ śreṇyaḥ praśāstā ca // (8.1)
caturthe viṣṭir nāyako mitrāmitrāṭavībalaṃ svapuruṣādhiṣṭhitam // (9.1)
vaṇijo rūpājīvāścānumahāpatham // (10.1)
bāhyato lubdhakaśvagaṇinaḥ satūryāgnayaḥ gūḍhāścārakṣāḥ // (11.1)
śatrūṇām āpāte kūpakūṭāvapātakaṇṭakinīśca sthāpayet // (12.1)
aṣṭādaśavargāṇām ārakṣaviparyāsaṃ kārayet // (13.1)
divāyāmaṃ ca kārayed apasarpajñānārtham // (14.1)
vivādasaurikasamājadyūtavāraṇaṃ ca kārayenmudrārakṣaṇaṃ ca // (15.1)
senānivṛttam āyudhīyam aśāsanaṃ śūnyapālo badhnīyāt // (16.1)
purastād adhvanaḥ samyakpraśāstā rakṣaṇāni ca / (17.1)
yāyād vardhakiviṣṭibhyām udakāni ca kārayet // (17.2)
