rājānam utthitam anūttiṣṭhante bhṛtyāḥ // (1.1)
pramādyantam anupramādyanti // (2.1)
karmāṇi cāsya bhakṣayanti // (3.1)
dviṣadbhiścātisaṃdhīyate / (4.1)
tasmād utthānam ātmanaḥ kurvīta // (4.2)
nālikābhir ahar aṣṭadhā rātriṃ ca vibhajet chāyāpramāṇena vā // (5.1)
tripauruṣī pauruṣī caturaṅgulā naṣṭacchāyo madhyāhneti catvāraḥ pūrve divasasyāṣṭabhāgāḥ // (6.1)
taiḥ paścimā vyākhyātāḥ // (7.1)
tatra pūrve divasasyāṣṭabhāge rakṣāvidhānam āyavyayau ca śṛṇuyāt // (8.1)
dvitīye paurajānapadānāṃ kāryāṇi paśyet // (9.1)
tṛtīye snānabhojanaṃ seveta svādhyāyaṃ ca kurvīta // (10.1)
caturthe hiraṇyapratigraham adhyakṣāṃśca kurvīta // (11.1)
pañcame mantripariṣadā pattrasampreṣaṇena mantrayeta cāraguhyabodhanīyāni ca budhyeta // (12.1)
ṣaṣṭhe svairavihāraṃ mantraṃ vā seveta // (13.1)
saptame hastyaśvarathāyudhīyān paśyet // (14.1)
aṣṭame senāpatisakho vikramaṃ cintayet // (15.1)
pratiṣṭhite 'hani saṃdhyām upāsīta // (16.1)
prathame rātribhāge gūḍhapuruṣān paśyet // (17.1)
dvitīye snānabhojanaṃ kurvīta svādhyāyaṃ ca // (18.1)
tṛtīye tūryaghoṣeṇa saṃviṣṭaścaturthapañcamau śayīta // (19.1)
ṣaṣṭhe tūryaghoṣeṇa pratibuddhaḥ śāstram itikartavyatāṃ ca cintayet // (20.1)
saptame mantram adhyāsīta gūḍhapuruṣāṃśca preṣayet // (21.1)
aṣṭame ṛtvigācāryapurohitasvastyayanāni pratigṛhṇīyāt cikitsakamāhānasikamauhūrtikāṃśca paśyet // (22.1)
savatsāṃ dhenuṃ vṛṣabhaṃ ca pradakṣiṇīkṛtyopasthānaṃ gacchet // (23.1)
ātmabalānukūlyena vā niśāharbhāgān pravibhajya kāryāṇi seveta // (24.1)
upasthānagataḥ kāryārthinām advārāsaṅgaṃ kārayet // (25.1)
durdarśo hi rājā kāryākāryaviparyāsam āsannaiḥ kāryate // (26.1)
tena prakṛtikopam arivaśaṃ vā gacchet // (27.1)
tasmād devatāśramapāṣaṇḍaśrotriyapaśupuṇyasthānānāṃ bālavṛddhavyādhitavyasanyanāthānāṃ strīṇāṃ ca krameṇa kāryāṇi paśyet kāryagauravād ātyayikavaśena vā // (28.1)
sarvam ātyayikaṃ kāryaṃ śṛṇuyān nātipātayet / (29.1)
kṛcchrasādhyam atikrāntam asādhyaṃ vāpi jāyate // (29.2)
agnyagāragataḥ kāryaṃ paśyed vaidyatapasvinām / (30.1)
purohitācāryasakhaḥ pratyutthāyābhivādya ca // (30.2)
tapasvināṃ tu kāryāṇi traividyaiḥ saha kārayet / (31.1)
māyāyogavidāṃ caiva na svayaṃ kopakāraṇāt // (31.2)
rājño hi vratam utthānaṃ yajñaḥ kāryānuśāsanam / (32.1)
dakṣiṇā vṛttisāmyaṃ tu dīkṣā tasyābhiṣecanam // (32.2)
prajāsukhe sukhaṃ rājñaḥ prajānāṃ ca hite hitam / (33.1)
nātmapriyaṃ hitaṃ rājñaḥ prajānāṃ tu priyaṃ hitam // (33.2)
tasmān nityotthito rājā kuryād arthānuśāsanam / (34.1)
arthasya mūlam utthānam anarthasya viparyayaḥ // (34.2)
anutthāne dhruvo nāśaḥ prāptasyānāgatasya ca / (35.1)
prāpyate phalam utthānāllabhate cārthasampadam // (35.2)
