śāsane śāsanam ityācakṣate // (1.1)
śāsanapradhānā hi rājānaḥ tanmūlatvāt saṃdhivigrahayoḥ // (2.1)
tasmād amātyasampadopetaḥ sarvasamayavid āśugranthaścārvakṣaro lekhanavācanasamartho lekhakaḥ syāt // (3.1)
so 'vyagramanā rājñaḥ saṃdeśaṃ śrutvā niścitārthaṃ lekhaṃ vidadhyāt deśaiśvaryavaṃśanāmadheyopacāram īśvarasya deśanāmadheyopacāram anīśvarasya // (4.1)
jātiṃ kulaṃ sthānavayaḥśrutāni karmarddhiśīlānyatha deśakālau / (5.1)
yaunānubandhaṃ ca samīkṣya kārye lekhaṃ vidadhyāt puruṣānurūpam // (5.2)
arthakramaḥ sambandhaḥ paripūrṇatā mādhuryam audāryaṃ spaṣṭatvam iti lekhasampat // (6.1)
tatra yathāvad anupūrvakriyā pradhānasyārthasya pūrvam abhiniveśa ityarthakramaḥ // (7.1)
prastutasyārthasyānuparodhād uttarasya vidhānam ā samāpter iti sambandhaḥ // (8.1)
arthapadākṣarāṇām anyūnātiriktatā hetūdāharaṇadṛṣṭāntair arthopavarṇanāśrāntapadateti paripūrṇatā // (9.1)
sukhopanītacāruarthaśabdābhidhānaṃ mādhuryam // (10.1)
agrāmyaśabdābhidhānam audāryam // (11.1)
pratītaśabdaprayogaḥ spaṣṭatvam iti // (12.1)
akārādayo varṇāstriṣaṣṭiḥ // (13.1)
varṇasaṃghātaḥ padam // (14.1)
taccaturvidhaṃ nāmākhyātopasarganipātāśceti // (15.1)
tatra nāma sattvābhidhāyi // (16.1)
aviśiṣṭaliṅgam ākhyātaṃ kriyāvāci // (17.1)
kriyāviśeṣakāḥ prādaya upasargāḥ // (18.1)
avyayāścādayo nipātāḥ // (19.1)
padasamūho vākyam arthaparisamāptau // (20.1)
ekapadāvaras tripadaparaḥ parapadārthānuparodhena vargaḥ kāryaḥ // (21.1)
lekhaparisaṃharaṇārtha itiśabdo vācikam asyeti ca // (22.1)
nindā praśaṃsā pṛcchā ca tathākhyānam athārthanā / (23.1)
pratyākhyānam upālambhaḥ pratiṣedho 'tha codanā // (23.2)
sāntvam abhyupapattiśca bhartsanānunayau tathā / (24.1)
eteṣvarthāḥ pravartante trayodaśasu lekhajāḥ // (24.2)
tatrābhijanaśarīrakarmaṇāṃ doṣavacanaṃ nindā // (25.1)
guṇavacanam eteṣām eva praśaṃsā // (26.1)
katham etad iti pṛcchā // (27.1)
evam ityākhyānam // (28.1)
dehi ityarthanā // (29.1)
na prayacchāmi iti pratyākhyānam // (30.1)
ananurūpaṃ bhavataḥ ityupālambhaḥ // (31.1)
mā kārṣīḥ iti pratiṣedhaḥ // (32.1)
idaṃ kriyatām iti codanā // (33.1)
yo 'haṃ sa bhavān yan mama dravyaṃ tad bhavataḥ ityupagrahaḥ sāntvam // (34.1)
vyasanasāhāyyam abhyupapattiḥ // (35.1)
sadoṣam āyatipradarśanam abhibhartsanam // (36.1)
anunayastrividho 'rthakṛtāvatikrame puruṣādivyasane ceti // (37.1)
prajñāpanājñāparidānalekhāstathā parīhāranisṛṣṭilekhau / (38.1)
prāvṛttikaśca pratilekha eva sarvatragaśceti hi śāsanāni // (38.2)
anena vijñāpitam evam āha tad dīyatāṃ ced yadi tattvam asti / (39.1)
rājñaḥ samīpe varakāram āha prajñāpanaiṣā vividhopadiṣṭā // (39.2)
bhartur ājñā bhaved yatra nigrahānugrahau prati / (40.1)
viśeṣeṇa tu bhṛtyeṣu tadājñālekhalakṣaṇam // (40.2)
yathārhaguṇasaṃyuktā pūjā yatropalakṣyate / (41.1)
apyādhau paridāne vā bhavatastāv upagrahau // (41.2)
jāter viśeṣeṣu pareṣu caiva grāmeṣu deśeṣu ca teṣu teṣu / (42.1)
anugraho yo nṛpater nideśāt tajjñaḥ parīhāra iti vyavasyet // (42.2)
nisṛṣṭisthāpanā kāryakaraṇe vacane tathā / (43.1)
eṣa vācikalekhaḥ syād bhaven naisṛṣṭiko 'pi vā // (43.2)
vividhāṃ daivasaṃyuktāṃ tattvajāṃ caiva mānuṣīm / (44.1)
dvividhāṃ tāṃ vyavasyanti pravṛttiṃ śāsanaṃ prati // (44.2)
dṛṣṭvā lekhaṃ yathātattvaṃ tataḥ pratyanubhāṣya ca / (45.1)
pratilekho bhavet kāryo yathā rājavacastathā // (45.2)
yatreśvarāṃścādhikṛtāṃśca rājā rakṣopakārau pathikārtham āha / (46.1)
sarvatrago nāma bhavet sa mārge deśe ca sarvatra ca veditavyaḥ // (46.2)
upāyāḥ sāmopapradānabhedadaṇḍāḥ // (47.1)
tatra sāma pañcavidhaṃ guṇasaṃkīrtanam sambandhopākhyānam parasparopakārasaṃdarśanam āyatipradarśanam ātmopanidhānam iti // (48.1)
tatrābhijanaśarīrakarmaprakṛtiśrutadravyādīnāṃ guṇagrahaṇaṃ praśaṃsā stutir guṇasaṃkīrtanam // (49.1)
jñātiyaunamaukhasrauvakulahṛdayamitrasaṃkīrtanaṃ sambandhopākhyānam // (50.1)
svapakṣaparapakṣayor anyonyopakārasaṃkīrtanaṃ parasparopakārasaṃdarśanam // (51.1)
asmin evaṃ kṛta idam āvayor bhavati ityāśājananam āyatipradarśanam // (52.1)
yo 'haṃ sa bhavān yan mama dravyaṃ tad bhavatā svakṛtyeṣu prayojyatām ityātmopanidhānam / (53.1)
iti // (53.2)
upapradānam arthopakāraḥ // (54.1)
śaṅkājananaṃ nirbhartsanaṃ ca bhedaḥ // (55.1)
vadhaḥ parikleśo 'rthaharaṇaṃ daṇḍaḥ / (56.1)
iti // (56.2)
akāntir vyāghātaḥ punaruktam apaśabdaḥ samplava iti lekhadoṣaḥ // (57.1)
tatra kālapattrakam acāruviṣam avirāgākṣaratvam akāntiḥ // (58.1)
pūrveṇa paścimasyānupapattir vyāghātaḥ // (59.1)
uktasyāviśeṣeṇa dvitīyam uccāraṇaṃ punaruktam // (60.1)
liṅgavacanakālakārakāṇām anyathāprayogo 'paśabdaḥ // (61.1)
avarge vargakaraṇaṃ cāvargakriyā guṇaviparyāsaḥ samplavaḥ / (62.1)
iti // (62.2)
sarvaśāstrāṇyanukramya prayogam upalabhya ca / (63.1)
kauṭilyena narendrārthe śāsanasya vidhiḥ kṛtaḥ // (63.2)
