upadhābhiḥ śuddhāmātyavargo gūḍhapuruṣān utpādayet kāpaṭikodāsthitagṛhapatikavaidehakatāpasavyañjanān sattritīkṣṇarasadabhikṣukīśca // (1.1)
paramarmajñaḥ pragalbhaśchāttraḥ kāpaṭikaḥ // (2.1)
tam arthamānābhyāṃ protsāhya mantrī brūyāt rājānaṃ māṃ ca pramāṇaṃ kṛtvā yasya yad akuśalaṃ paśyasi tat tadānīm eva pratyādiśa iti // (3.1)
pravrajyāpratyavasitaḥ prajñāśaucayukta udāsthitaḥ // (4.1)
sa vārttākarmapradiṣṭāyāṃ bhūmau prabhūtahiraṇyāntevāsī karma kārayet // (5.1)
karmaphalācca sarvapravrajitānāṃ grāsācchādanāvasathān pratividadhyāt // (6.1)
vṛttikāmāṃścopajapet etenaiva veṣeṇa rājārthaścaritavyo bhaktavetanakāle copasthātavyam iti // (7.1)
sarvapravrajitāśca svaṃ svaṃ vargam evam upajapeyuḥ // (8.1)
karṣako vṛttikṣīṇaḥ prajñāśaucayukto gṛhapatikavyañjanaḥ // (9.1)
sa kṛṣikarmapradiṣṭāyāṃ bhūmau iti samānaṃ pūrveṇa // (10.1)
vāṇijako vṛttikṣīṇaḥ prajñāśaucayukto vaidehakavyañjanaḥ // (11.1)
sa vaṇikkarmapradiṣṭāyāṃ bhūmau iti samānaṃ pūrveṇa // (12.1)
muṇḍo jaṭilo vā vṛttikāmastāpasavyañjanaḥ // (13.1)
sa nagarābhyāśe prabhūtamuṇḍajaṭilāntevāsī śākaṃ yavamuṣṭiṃ vā māsadvimāsāntaraṃ prakāśam aśnīyāt gūḍham iṣṭam āhāram // (14.1)
vaidehakāntevāsinaścainaṃ samiddhayogair arcayeyuḥ // (15.1)
śiṣyāścāsyāvedayeyuḥ asau siddhaḥ sāmedhikaḥ iti // (16.1)
samedhāśāstibhiścābhigatānām aṅgavidyayā śiṣyasaṃjñābhiśca karmāṇyabhijane avasitānyādiśet alpalābham agnidāhaṃ corabhayaṃ dūṣyavadhaṃ tuṣṭidānaṃ videśapravṛttijñānam idam adya śvo vā bhaviṣyati idaṃ vā rājā kariṣyati iti // (17.1)
tad asya gūḍhāḥ sattriṇaśca sampādayeyuḥ // (18.1)
sattvaprajñāvākyaśaktisampannānāṃ rājabhāgyam anuvyāharenmantrisaṃyogaṃ ca brūyāt // (19.1)
mantrī caiṣāṃ vṛttikarmabhyāṃ viyateta // (20.1)
ye ca kāraṇād abhikruddhāstān arthamānābhyāṃ śamayet akāraṇakruddhāṃstūṣṇīṃ daṇḍena rājadviṣṭakāriṇaśca // (21.1)
pūjitāścārthamānābhyāṃ rājñā rājopajīvinām / (22.1)
jānīyuḥ śaucam ityetāḥ pañcasaṃsthāḥ prakīrtitāḥ // (22.2)
