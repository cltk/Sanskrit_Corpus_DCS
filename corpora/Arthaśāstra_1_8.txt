
Appointment of ministers
sahādhyāyino 'mātyān kurvīta dṛṣṭaśaucasāmarthyatvāt iti bhāradvājaḥ // (1.1)
te hyasya viśvāsyā bhavanti iti // (2.1)
neti viśālākṣaḥ // (3.1)
sahakrīḍitatvāt paribhavantyenam // (4.1)
ye hyasya guhyasadharmāṇastān amātyān kurvīta samānaśīlavyasanatvāt // (5.1)
te hyasya marmajñabhayān nāparādhyanti iti // (6.1)
sādhāraṇaiṣa doṣaḥ iti pārāśarāḥ // (7.1)
teṣām api marmajñabhayāt kṛtākṛtānyanuvarteta // (8.1)
yāvadbhyo guhyam ācaṣṭe janebhyaḥ puruṣādhipaḥ / (9.1)
avaśaḥ karmaṇā tena vaśyo bhavati tāvatām // (9.2)
yainam āpatsu prāṇābādhayuktāsvanugṛhṇīyustān amātyān kurvīta dṛṣṭānurāgatvāt iti // (10.1)
neti piśunaḥ // (11.1)
bhaktir eṣā na buddhiguṇaḥ // (12.1)
saṃkhyātārtheṣu karmasu niyuktā ye yathādiṣṭam arthaṃ saviśeṣaṃ vā kuryustān amātyān kurvīta dṛṣṭaguṇatvāt iti // (13.1)
neti kauṇapadantaḥ // (14.1)
anyair amātyaguṇair ayuktā hyete // (15.1)
pitṛpaitāmahān amātyān kurvīta dṛṣṭāvadānatvāt // (16.1)
te hyenam apacarantam api na tyajanti sagandhatvāt // (17.1)
amānuṣeṣvapi caitad dṛśyate // (18.1)
gāvo hyasagandhaṃ gogaṇam atikramya sagandheṣvevāvatiṣṭhante iti // (19.1)
neti vātavyādhiḥ // (20.1)
te hyasya sarvam avagṛhya svāmivat pracaranti // (21.1)
tasmān nītivido navān amātyān kurvīta // (22.1)
navāstu yamasthāne daṇḍadharaṃ manyamānā nāparādhyanti iti // (23.1)
neti bāhudantīputraḥ // (24.1)
śāstravid adṛṣṭakarmā karmasu viṣādaṃ gacchet // (25.1)
tasmād abhijanaprajñāśaucaśauryānurāgayuktān amātyān kurvīta guṇaprādhānyāt iti // (26.1)
sarvam upapannam iti kauṭilyaḥ // (27.1)
kāryasāmarthyāddhi puruṣasāmarthyaṃ kalpyate // (28.1)
sāmarthyataśca vibhajyāmātyavibhavaṃ deśakālau ca karma ca / (29.1)
amātyāḥ sarva evaite kāryāḥ syur na tu mantriṇaḥ // (29.2)
