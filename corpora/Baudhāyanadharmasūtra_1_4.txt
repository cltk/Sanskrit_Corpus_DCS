 dharmārthau yatra na syātāṃ śuśrūṣā vāpi tadvidhā / (1.1)
vidyayā saha martavyaṃ na cainām ūṣare vapet // (1.2)
agnir iva kakṣaṃ dahati brahma pṛṣṭam anādṛtam / (2.1)
tasmād vai śakyaṃ na brūyād brahma mānam akurvatām iti // (2.2)
evāsmai vaco vedayante // (3.1)
brahma vai mṛtyave prajāḥ prāyacchat tasmai brahmacāriṇameva na prāyacchat so 'bravīd astu mahyam apy etasmin bhāga iti yām eva rātriṃ samidhaṃ nāharātaīti // (4.1)
tasmād brahmacārī yāṃ rātriṃ samidhaṃ nāharaty āyuṣa eva tām avadāya vasati tasmād brahmacārī samidham āharen ned āyuṣo 'vadāya vasānīti // (5.1)
dīrghasattraṃ vā eṣa upaiti yo brahmacaryam upaiti sa yāmupayan samidham ādadhāti sā prāyaṇīyātha yāṃ snāsyansodayanīyātha yā antareṇa sattryā evāsya tāḥ // (6.1)
brāhmaṇo vai brahmacaryam upayan caturdhā bhūtāni praviśaty agniṃ padā mṛtyuṃ padācāryaṃ padātmany evāsya caturthaḥ pādaḥ pariśiṣyate / (7.1)
sa yad agnau samidham ādadhāti ya evāsyāgnau pādas tameva tena parikrīṇāti taṃ saṃskṛtyātman dhatte sa enam āviśati / (7.2)
atha yad ātmānaṃ daridrīkrityāhrīr bhūtvā bhikṣate brahmacaryaṃ carati ya evāsya mṛtyau pādas tam eva tena parikrīṇāti taṃ saṃskṛtyātman dhatte sa enam āviśati / (7.3)
atha yad ācāryavacaḥ karoti ya evāsyācārye pādas tameva tena parikrīṇāti taṃ saṃskṛtyātman dhatte sa enam āviśati / (7.4)
atha yat svādhyāyam adhīte ya evāsyātmani pādas tameva tena parikrīṇāti taṃ saṃskṛtyātman dhatte sa enam āviśati / (7.5)
na ha vai snātvā bhikṣeta api ha vai snātvā bhikṣāṃ caraty api jñātīnām aśanāyāpi pitṝṇām anyābhyaḥ kriyābhyaḥ / (7.6)
sa yad anyāṃ bhikṣitavyāṃ na vindetāpi svāmevācāryajāyāṃ bhikṣetātho svāṃ mātaram / (7.7)
nainaṃ saptamy abhikṣitātīyāt // (7.8)
bhaikṣasyācaraṇe doṣaḥ pāvakasyāsamindhane / (8.1)
saptarātram akṛtvaitad avakīrṇivrataṃ caret // (8.2)
tam evaṃ vidvāṃsam evaṃ carantaṃ sarve vedā āviśanti // (9.1)
yathā ha vā agniḥ samiddho rocata evaṃ ha vā eṣa snātvā rocate ya evaṃ vidvān brahmacaryaṃ caratīti brāhmaṇam iti brāhmaṇam // (10.1)
