 atha kamaṇḍalucaryām upadiśanti // (1.1)
chāgasya dakṣiṇe karṇe pāṇau viprasya dakṣiṇe / (2.1)
apsu caiva kuśastambe pāvakaḥ paripaṭhyate // (2.2)
tasmācchaucaṃ kṛtvā pāṇinā parimṛjīta paryagnikaraṇaṃ hi tat uddīpyasva jātaveda iti punardāhād viśiṣyate // (3.1)
tatrāpi kiṃcit saṃspṛṣṭaṃ manasi manyeta kuśairvā tṛṇair vā prajvālya pradakṣiṇaṃ paridahanam // (4.1)
ata ūrdhvaṃ śvavāyasaprabhṛtyupahatānām agnivarṇa ityupadiśanti // (5.1)
mūtrapurīṣalohitaretaḥprabhṛtyupahatānām utsargaḥ // (6.1)
bhagne kamaṇḍalau vyāhṛtibhiḥ śataṃ juhuyāj japed vā // (7.1)
bhūmir bhūmim agān mātā mātaram apy agāt bhūyāsma putraiḥ paśubhir yo no dveṣṭi sa bhidyatām iti // (8.1)
kapālāni saṃhṛtyāpsu prakṣipya sāvitrīṃ daśāvarāṃ kṛtvā punar evānyaṃ gṛhṇīyāt // (9.1)
varuṇam āśritya etat te varuṇa punar eva mām om iti akṣaraṃ dhyāyet // (10.1)
śūdrād gṛhya śataṃ kuryād vaiśyād ardhaśataṃ smṛtam / (11.1)
kṣatriyāt pañcaviṃśat tu brāhmaṇād daśa kīrtitāḥ // (11.2)
astamita āditya udakaṃ gṛhṇīyān na gṛhṇīyād iti mīmāṃsante brahmavādinaḥ // (12.1)
gṛhṇīyād ity etad aparam // (13.1)
yāvad udakaṃ gṛhṇīyāt tāvat prāṇam āyacchet // (14.1)
agnir ha vai hy udakaṃ gṛhṇāti // (15.1)
kamaṇḍalūdakenābhiṣiktapāṇipādo yāvad ārdraṃ tāvadaśuciḥ pareṣām ātmānam eva pūtam karoti nānyat karma kurvīteti vijñāyate // (16.1)
api vā pratiśaucam ā maṇibandhācchucir iti baudhāyanaḥ // (17.1)
athāpy udāharanti // (18.1)
